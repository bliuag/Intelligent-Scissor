The Intelligent Scissor

Presented by LIU Boyu and YAN Rui for COMP 5421 in Spring, 2016.

How to use?

- CTRL+'[': Zoom out

- CTRL+']': Zoom in

- CTRL+'o': Open

- CTRL+'c': Save Contour

- CTRL+'m': Save Mask

- CTRL+'q': Quit

- CTRL+'b': Brush

- CTRL+'s': Scissor

- '.': Finish Contour

- CTRL+'.': Finish Closed Contour

- CTRL+'z': Undo

Notes

- If brush is needed, press Edit --> Scissor then drag along with the mouse on the canvas.

- Press Edit-->Scissor in order to start scissoring(after using the brush or not).

- After using brush and scissor, when clicking Image Only, the original image will be shown while after clicking Image with Contour, the canvas with brush and contour will appear again.

- The Path Tree and Min Path function can only be used with at least one seed is settled.

Extra credit features

- The brush function is implemented as instructions.

- Blur in 3x3, 5x5 and 7x7 pixel.

Observation: It will makes the selections of hard edges more difficult, while enables the delicate part (e.g hair and fur) easier to select.
